# RentalCarApplication
This is simple rental car app which base on WPF C#.
# Patterns:
- MVVM;
- Command;
- Repository;
- Unit of work;
- Abstract Factory;
# ORM-model: 
- Entity Framework Core with Code First for MS SQL;
- Encryption of user password with MD5.
# Design:
- Material Design Library.
